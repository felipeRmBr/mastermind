Mastermind is a strategy game of Codemaker vs. Codebreaker.

The Codemaker starts by choosing a pattern of four code pegs; duplicates and blanks are allowed depending on game settings, so the player could even choose four code pegs of the same color or four blanks. Then, the Codebraker should try to use logic and deduction to guess the pattern, in both order and color.

Each guess is made by placing a column of code pegs on the decoding board. Once the guess is in place, the Codemaker provides feedback by placing from zero to four marbles in the small holes of the column with the guess. A red marble is placed for each code peg from the guess which is correct in both color and position; and a white marble is placed for each code peg of the right color placed in the wrong position.

In the single-player mode you will play the Codebreaker, and the MasterBot will play the Codemaker. At each game you earn points depending on the number of guesses you make before braking the code. You can win up to 10 points, but if after ten guesses you fail to brake the code then you lose 5 points.
